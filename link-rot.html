Link rot

Putrescerum catena

As organic matter decays away through physical consumption, so to does digital information. Link rot occurs when files move from their original location, causing links to those files to become unusable. If this page consists of only text and a gif, as opposed to an interactive 3D model, then this website has been eaten by link rot bacteria.
